HTML, siglas de Hyper Text Markup Language, hace referencia al lenguaje de marcado para la elaboración de páginas web. Es un estándar que save de referencia para la elaboración de páginas web en sus liferentes versiones, define una estructura basica y un código denominado código HTML) para la definición de contenido de una página web, come texto, imágenes, entre otros. Es un estándar a cargo de la W3C. organizacion dedicada a la estandarización de casi todas las tecnologias ligadas a la web, sobre todo en lo referente a su escritura e interpretación. Es el lenguaje con el que se definen las páginas web.
